Run the standard implementation loop for {task}:

1. **scout** — explore codebase, map relevant files, find entry points
2. **planner** — create concrete implementation plan from scout findings
3. **worker** — implement the plan (async)
4. **code-reviewer** + **architect** — parallel review (fresh context)
5. **fixer** — apply accepted fixes
6. **validator** — verify tests pass and acceptance criteria met
